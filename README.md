Перер Вами - "Доска объявлений", написанная в фреймворке Django. В данном проекте используется рендеринг шаблонов, авторизация и регистрация пользователей.
Можно создавать новых пользователей, добавлять новые объявления и просматривать существующие.

Для запуска проекта после установки зависимостей с помощью pip install -г requirements.txt нужно перейти в директорию urban_project (cd urbanproject) и запустить проект командой "python manage.ру runserver"
